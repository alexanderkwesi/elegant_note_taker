# Elegant Note Taker

A lightweight note-taking app for creating and organising simple notes in the browser.

## Why I Built It

I built this project to practise CRUD-style user interactions, local interface state and simple productivity app design.

## Features

- Browser-based note-taking interface
- HTML, CSS and JavaScript structure
- Node package script for opening the app
- Simple productivity workflow

## Tech Stack

- HTML
- CSS
- JavaScript
- Node package tooling

## Project Structure

```text
index.html
script.js
styles.css
package.json
README.md
```

## Getting Started

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

## What This Shows

This project demonstrates frontend interaction logic, simple CRUD thinking and clean UI organisation for a productivity tool.
